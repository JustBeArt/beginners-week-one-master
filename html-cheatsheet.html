HTML

HTML is a structure of content (HYPER TEXT MARKUP)

To have a valide HTML document, we need to tell the browser we're using HTML5. Every HTML5 document requires this layout:

Exemple

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>


Html documents use TAGS < >

  • 'h' tag for headings, they are ordered from largest to smallest size: h1, h2, h3, h4, h5, h6
  • 'div' element is short for "division" or a container that divides the page into section(Great for grouping elements!)
  • 'p' tag for paragraph
  • 'body' tag wraps around all the content that's displayed on the page
  • before 'body', we always start with 'head'; It wraps around the 'style' tag and other elements tahat are not content on the page itself, such as the 'title' tag
  • The 'title' tag controls what shows up in the browser tab
  • 'style' tag is used to define style information for HTML document
  • 'img' tag is used to insert an image, img tags are like br tags : they don't come in pairs. The ' alt property should be used to provide a description of the image in case it won't load or the user is partialy sighted
  • 'div' tag is nothing more than a container unit that encapsulates other page elements and divides the HTML document into sections. Web developers use 'div' elements to group together HTML elements and apply CSS styles to many elements at once
  • 'br' tag produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant. The br tag is a special type of tag(called a 'void tag') you don't need a closing br tag, just an opening one
  • 'ol' element displays numbered list
  • 'ul' element used to designate an unordered list. It is useful for creating bulleted lists, those in which the order of the items is arbitrary.
  • tag represents a list item in ordered and unordered lists. The 'li' tag is placed inside either a 'ol' tag or a 'ul' to represent each individual item within that list.
  • 'em' tag marks text that has stress emphasis which traditionally means that the text is displayed in italics by the browser.
